A smart AI-powered Mood Based Learning Website with mood detection, study planner, mock tests, notes library, analytics dashboard, motivation tools, and productivity features for students.


# 🧠 Mood Based Learning Website

A smart and interactive learning platform designed to improve student productivity based on mood, focus level, and study habits.

---

# 🚀 Live Features

## 🎯 Core Modules

- Mood Detection using Camera
- AI Recommendation Box
- Study Planner with Timer
- Daily Goal Saving
- Music During Study
- Progress Tracker
- Motivation Quotes
- Subject Wise Mock Test
- Smart Food Suggestions
- Digital Notes Library
- Analytics Dashboard
